%---------- Abstract ----------
\begin{abstract}
We consider the synthesis of sketches for constraints specified using logical formulae using deep learning techniques. For this problem in program synthesis, we discuss relevant literature, gaps in existing techniques and our approach to attack the problem. This report broadly consists of a research proposal and preliminary progress in this direction. For the preliminary direction,  we discuss a prototype tool as a work in progress.

\end{abstract}
